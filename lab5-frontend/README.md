# Lab 5 Frontend — Summarization UI

Text-In → Summary-Out web interface for the Lab 4 FastAPI summarization backend.

## Requirements

- Node.js 18 or higher
- npm

## Setup

1. Install Node.js from https://nodejs.org (LTS) if not already installed.

2. Install dependencies:

   ```bash
   cd lab5-frontend
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open http://localhost:3000 in your browser.

## Backend

Ensure the Lab 4 backend is running at http://localhost:8000 with the `POST /summarize` endpoint.
